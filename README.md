## English

### Overview

SoundGen is a web-based sound generator application that allows users to create, manipulate, and explore different audio effects and tones directly in their web browser.

### Features

- 🎼 **Multiple Waveform Support**: Generate sine, square, sawtooth, and triangle waves
- 🔊 **Real-time Audio Synthesis**: Instantly create and listen to generated sounds
- 🎚️ **Frequency Control**: Adjust frequency to create different pitches and tones
- 📊 **Volume Control**: Manage output volume with precision
- 🎨 **User-Friendly Interface**: Intuitive controls for easy sound experimentation
- ⚡ **No Installation Required**: Works directly in your web browser
- 🚀 **Lightweight**: Fast loading and responsive performance

### Quick Start

1. Clone this repository:
```bash
git clone https://github.com/majiangla/SoundGen.git
cd SoundGen
```

2. Open `index.html` in your web browser

3. Use the controls to:
   - Select waveform type
   - Adjust frequency (pitch)
   - Control volume
   - Play/Stop sounds

### Usage

1. **Select Waveform**: Choose from sine, square, sawtooth, or triangle waves
2. **Set Frequency**: Adjust the frequency slider to change the pitch (in Hz)
3. **Control Volume**: Set the desired volume level
4. **Play Sound**: Click the Play button to generate the sound
5. **Stop Sound**: Click the Stop button to end playback

### Technology Stack

- **HTML5**: Structure and Web Audio API
- **CSS3**: Modern styling and responsive layout
- **JavaScript (ES6+)**: Audio synthesis and real-time control

### Requirements

- Modern web browser with Web Audio API support
- No external dependencies or frameworks

### Browser Compatibility

Compatible with all modern browsers supporting:
- HTML5 Web Audio API
- CSS3 Flexbox/Grid
- ES6 JavaScript features

Tested on:
- Chrome/Chromium 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 中文

### 项目介绍

声音发生器（SoundGen）是一款基于网页的音频生成工具，用户可以直接在浏览器中创建、操作和探索各种音效和声调。

### 主要特性

- 🎼 **多种波形支持**：生成正弦波、方波、锯齿波和三角波
- 🔊 **实时音频合成**：即时生成和播放声音
- 🎚️ **频率控制**：调整频率以创建不同的音调和音高
- 📊 **音量控制**：精确管理输出音量
- 🎨 **用户友好的界面**：直观的控制面板，易于使用
- ⚡ **无需安装**：直接在浏览器中运行
- 🚀 **轻量级应用**：快速加载，响应迅速

### 快速开始

1. 克隆此仓库：
```bash
git clone https://github.com/majiangla/SoundGen.git
cd SoundGen
```

2. 在浏览器中打开 `index.html`

3. 使用控制面板：
   - 选择波形类型
   - 调整频率（音高）
   - 控制音量
   - 播放/停止声音

### 使用说明

1. **选择波形**：从正弦波、方波、锯齿波或三角波中选择
2. **设置频率**：调整频率滑块来改变音调（单位：赫兹/Hz）
3. **控制音量**：设置所需的音量大小
4. **播放声音**：点击播放按钮生成声音
5. **停止播放**：点击停止按钮结束播放

### 技术栈

- **HTML5**：页面结构和 Web Audio API
- **CSS3**：现代化样式和响应式布局
- **JavaScript (ES6+)**：音频合成和实时控制

### 系统要求

- 支持 Web Audio API 的现代浏览器
- 无需外部依赖或框架

### 浏览器兼容性
